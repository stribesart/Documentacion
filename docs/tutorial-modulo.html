Paso 1: Definir el nombre de nuestro modulo

Para nuestro primer paso en la creacion de nuestro modulo necesitamos definir el nombre del mismo a crear en este caso se nos pide llamarlo Uniformes.
Se necesita navegar en el proyecto comercialPruebas hasta encontrar el archivo llamado Menu.html el cual tiene una estructura como la siguiente.

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>PRADA Comercial</title>
  </head>
  <body>
    <nav class="navbar navbar-expand-lg navbar-light" id="menu-principal"></nav>
    <main class="container"></main>
    <footer></footer>
  </body>
</html>

Dentro de esta estructura tenemos que navegar hasta llegar a la etiqueta nav que es la que contiene los botones de nuestra pagina mostrada anteriormente.

Paso 2: Construir los elementos de Uniformes

Una vez creado el nuevo boton se tendra que crear una carpeta dentro de la carpeta publico dentro de la carpeta js nombrada de la misma manera que el boton, en este caso crearemos nuestra carpeta Uniformes.
Una vez creada la carpeta podemos comenzar el desarrollo de nuestra pagina.
Crearemos un nuevo documento llamandolo Uniformes.js y un archivo llamado FiltrosUniformes.js.
Al inicio de nuestro archivo Uniformes.js crearemos la funcion que nos exportara nuestra pagina ya construida, de la siguiente manera:

export default async function Uniformes() {}

Ahora ya tenemos un modulo creado dentro de nuestra pagina. Para esto necesitamos saber que nuestra pagina principal se compone de sus filtros y su contenido, entonces comenzaremos con los filtros.
